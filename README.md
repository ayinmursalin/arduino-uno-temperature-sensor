This project is created as my journey on Learning IoT.

## Description
Things
- Arduino Uno
- LCD with i2c
- DHT-11 Module

DHT-11 is a module to read Temperature and Humidity,
This module has 3 pin:
- VCC
- GND
- Out (Digital)
